# Robleto

This is the 2022 design of the personal website for Greg Robleto. This site is a small site to share information about  skills, interests and means of contacting Greg.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.


### Installing

This site is HTML, CSS and Javascript and has no requirements for installation of a virtual environment.

## Deployment

This site is hosted by Netlify. Basic setup is all that was required.

## Built Using

* [Photoshop](http:www.adobe.com/products/photoshop.html) - For creating the background and portfolio images
* [VS Code](https://code.visualstudio.com) - For coding the HTML, CSS and Javascript as well as the JSON and Markdown for this Readme
* [11ty](https://www.11ty.dev/) - As a framework for building and delopying the static site.
* [Netlify](https://www.netlify.com) - Used for building and hosting the site.
