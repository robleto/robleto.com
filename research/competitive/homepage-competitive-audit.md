## High-Impact Deltas for robleto.com

### Content/Structure

- `Finding`: This peer class is universally writing-forward while also surfacing work entry points.
  - `Evidence`: writing **20/20**; work/case artifacts **14/20**.
  - `Target delta`: writing is strong on `robleto.com`; top-level work path can be made more explicit in hero flow.
  - `Recommendation`: keep writing prominence, and elevate one “start with selected work” path directly under hero copy.
  - `Type`: content/structure

- `Finding`: Explicit services framing is not universal, but it is a meaningful differentiator when present.
  - `Evidence`: services positioning **7/20**.
  - `Target delta`: current positioning is capability-led but still implicit.
  - `Recommendation`: add a compact “How I help” block with 2-4 collaboration modes.
  - `Type`: content/structure

- `Finding`: Contact path visibility is common among peers.
  - `Evidence`: contact present **15/20**.
  - `Target delta`: contact exists in nav, but could be integrated earlier in decision flow.
  - `Recommendation`: add a short contact/response promise line in upper-page proof rail.
  - `Type`: content/structure

### Visual/Layout

- `Finding`: Trust is usually conveyed through role history, long-running writing, and artifact breadth rather than testimonials.
  - `Evidence`: testimonials **0/20**, but about **20/20** and writing **20/20**.
  - `Target delta`: trust cues on `robleto.com` are distributed.
  - `Recommendation`: cluster trust cues near top (role context + selected artifacts + quick contact path).
  - `Type`: visual/layout

- `Finding`: Metrics/logos are relatively rare in this peer subset.
  - `Evidence`: metrics **3/20**, logos **2/20**.
  - `Target delta`: missing metrics/logos is not a critical deficiency in this cohort.
  - `Recommendation`: treat metrics/logos as optional enhancements, not first-order requirements.
  - `Type`: visual/layout

## Differentiator Patterns to Preserve

- Keep your exploratory modules (`Signals`, `Writing`, `Influences`) as a core identity asset.
- Preserve the personal-web feel; improve conversion clarity without “agency-ifying” the homepage.

## Recommended Next Experiments

1. Add a compact **How I help** section under hero. *(Impact: High, Effort: Low)*
2. Add a top **Start here / Selected work** CTA cluster. *(Impact: High, Effort: Low)*
3. Add a small **proof rail** (role context + recency + contact promise). *(Impact: Medium, Effort: Low)*
4. Reframe writing intro as **method + proof**, not only publication. *(Impact: Medium, Effort: Low)*
5. Optionally add **Now** or **newsletter** if audience recurrence is a goal. *(Impact: Medium, Effort: Medium)*
