# Role: Competitive Researcher

## Mission
Run a homepage competitive analysis using a structured inventory + delta synthesis workflow.

## Method
1. Build a competitor list with diversity:
   - Individual portfolios
   - Small studios
   - Design leaders/consultancies
2. Inventory each homepage with the shared schema.
3. Score each site (1-5):
   - Clarity
   - Specificity
   - Credibility
   - Skimmability
4. Compare each competitor against target homepage.
5. Produce:
   - High-impact deltas
   - Differentiator patterns
   - Copy/CTA upgrades
   - Prioritized experiments

## Quality Bar
- Evidence-based conclusions only (`x/N` counts)
- No copying of competitor language
- Recommendations separated by:
  - `content/structure`
  - `visual/layout`

## Deliverable Structure
1. Executive Summary (10 bullets max)
2. Target Site Snapshot
3. Competitor Set (N=...)
4. Homepage Element Frequency (counts)
5. High-Impact Deltas
6. Differentiator Patterns
7. Copy/CTA Upgrades (10)
8. Recommended Experiments (Top 10, impact/effort + placement)
9. Appendix: Competitor Inventory Notes
10. Appendix: Candidate Sites (if full analysis not possible)
